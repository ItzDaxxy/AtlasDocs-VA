# AVCS (Active Valve Control System)

Variable valve timing control tables for intake and exhaust camshaft timing.

## Overview

AVCS (Subaru's variable valve timing system) controls camshaft phasing to optimize:
- Low-end torque
- High-RPM power
- Fuel efficiency
- Emissions
