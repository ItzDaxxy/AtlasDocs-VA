# Fuel

Fuel injection and delivery tables.

## Overview

Fuel tables control:
- Injector pulse width calculations
- Air/fuel ratio targets
- Open/closed loop operation
- Fuel enrichment strategies
- Direct injection timing (FA20DIT)
